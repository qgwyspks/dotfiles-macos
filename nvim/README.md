## 配置结构

```dir
.
├── README.md
├── init.lua                 -- 配置入口
└── lua/                     -- LUA 配置目录
    ├── core/                -- 设置目录
    │   ├── autocmds.lua     -- 自动命令
    │   ├── keymaps.lua      -- 快捷键
    │   ├── lazy.lua         -- Lazy.nvim 安装
    │   └── options.lua      -- 常规配置
    └── plugins/             -- 插件配置目录
        ├── colorscheme.lua  -- 主题
        ├── configs/         -- 各个插件的配置目录
        ├── editor.lua       -- 代码工具
        ├── lsp.lua          -- 代码补全
        ├── ui.lua           -- ui 界面
        └── util.lua         -- 工具
```

## 快捷键

| 模式   | 键                 | 说明 |
| ------ | ------------------ | ---- |
| normal | `<leader>` + ww |   |
| normal | `<leader>` + wd | 删除当前窗口（不能删除最后一个） |
| normal | `<leader>` + -  | 水平分屏 |
| normal | `<leader>` + \  | 垂直分屏 |
| normal | Ctrl + t  | 打开或关闭文件树 |
| normal | `<leader>` + t | 打开或关闭文件树 |
| normal | `<leader>` + tf | 聚焦到文件树上 |
| normal | `<leader>` + tc | 折叠文件树 |
| normal | `<leader>` + bp |  |
| normal | `<leader>` + mv | 打开或关闭 Markdown Preview |
| normal | `<leader>` + sf | 打开 Telescope 搜索文件 |
